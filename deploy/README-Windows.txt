DST DS Panel - Windows
======================

1. Extract all files to a folder (e.g. C:\DST-DS-Panel)

2. Double-click "dst-ds-panel-tray.exe"
   - A "DST" icon appears in the system tray (bottom-right)
   - Click it -> Start Server -> Open Panel

3. In the web panel:
   - Click "Install DST" to download the dedicated server
   - Create a cluster, set your token, and start!

No Docker required - DST runs as a native process.

Data is stored in: %LOCALAPPDATA%\DST DS Panel\
