DST DS Panel - Installation
===========================

1. Drag "DST DS Panel" to the Applications folder →

2. If macOS shows "damaged" or "can't be opened" warning,
   open Terminal and run:

   xattr -cr "/Applications/DST DS Panel.app"

3. Double-click to launch from Applications.
   The app runs in the menu bar (top-right of screen).

4. Click the "DST" menu bar icon → Start Server → Open Panel

Requirements: Docker Desktop or OrbStack, Homebrew
